# British Isles Adventures

Planning reference for a family nature-first holiday across the UK and Ireland — national parks, activities, lodging, transport, gear, and booking triage — plus a small web app for picking and ranking favorite activities.

**Trip style:** nature-first, rustic villages & countryside, calm/uncrowded, family with teenagers, mix of relaxation and adventure activities, UK with an optional Ireland leg.

## Reference cards

| File | Covers |
|---|---|
| [`1-activities.md`](1-activities.md) | Top destination bases (Lake District, Snowdonia, Pembrokeshire, Cairngorms, Skye, etc.) and 296+ activities across 17 categories, with cost estimates. |
| [`2-lodging.md`](2-lodging.md) | Lodging types (cottages, farm stays, glamping, narrowboats, castles...) rated on feel, accessibility, and family-friendliness, with per-night costs. |
| [`3-logistics.md`](3-logistics.md) | Car rental vs. public transport by destination, ferries, and parking realities. |
| [`4-gear.md`](4-gear.md) | Packing list and seasonal (August) considerations — weather, midges, daylight, crowds. |
| [`5-bookings.md`](5-bookings.md) | Booking lead-time triage — what to book immediately vs. what to treat as a stretch goal. |

All costs are approximate 2025–2026 guide prices (£ GBP, € EUR for Ireland-specific items) — treat as planning ballparks, not quotes.

Supporting file: [`-activities-list-for-linkmatrix.txt`](-activities-list-for-linkmatrix.txt) — flat list of activity names.

## Adventure Picks app

[`adventure-picks-app/`](adventure-picks-app/) is a single-page vanilla JS/HTML/CSS app for picking, ranking, and sharing favorite activities sourced from `1-activities.md`. No dependencies, no build step. See [`adventure-picks-app/README.md`](adventure-picks-app/README.md) for how to run it and how the data/app flow works.
