# British Isles Adventures

Planning docs and a small web app for a family nature-focused UK & Ireland holiday — picking destinations, lodging, activities, gear, and bookings, then letting everyone vote on and rank their favorite activities.

## Planning reference cards

| File | Covers |
|---|---|
| [`1-activities.md`](1-activities.md) | Destination bases (tiered) plus activity tables by category, with cost estimates |
| [`2-lodging.md`](2-lodging.md) | Lodging types (cottages, farm stays, glamping, etc.) rated on feel, accessibility, family-friendliness, cost |
| [`3-logistics.md`](3-logistics.md) | Transport — car needs per destination, ferries, parking |
| [`4-gear.md`](4-gear.md) | Packing list and seasonal (August) considerations |
| [`5-bookings.md`](5-bookings.md) | Booking lead times and what's realistically still bookable close to departure |
| [`-activities-list-for-linkmatrix.txt`](-activities-list-for-linkmatrix.txt) | Flat activity list for cross-referencing |

All cost figures are approximate 2025–2026 guide prices (£ for UK, € noted for Ireland) — treat as planning ballparks, not quotes.

## adventure-picks-app

A single-page vanilla JS/HTML/CSS app for picking, ranking, and sharing favorite activities sourced from `1-activities.md`. See [`adventure-picks-app/README.md`](adventure-picks-app/README.md) for how to run it and how the data/app flow work.
